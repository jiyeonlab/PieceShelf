# 📚Ribrary

### 내가 본 영화, 책, 전시회 등을 기록하기 위한 앱입니다 :)

### 기능

- 웹 검색, 앨범에서 선택, 카메라 촬영을 통해 이미지 추가 가능
- 설정 탭에서 카테고리 추가 및 삭제 가능
- 리스트에서 이미지를 선택하면, 입력한 날짜와 메모 등의 기록 확인 가능

### 사용한 기술 및 라이브러리

- Swift
- Alamofire
- 네이버 검색 API, Firebase Database, Firebase Storage
- AutoLayout

### 사용한 아키텍쳐

- MVC 기반
- Delegate, Singleton, Notification

### 진행기간
